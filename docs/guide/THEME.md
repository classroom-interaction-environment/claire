# CLAIRE Theme guide

## Buttons

All Buttons are `primary` except those for the following action types:

- create - `success`
- delete - `danger`
- remove - `danger`
- add/select - `secondary`
- cancel - `secondary`
- invite - `secondary`
- start - `success`
- complete - `secondary`

All buttons are filled, except those in list-view, which are outlined only.
